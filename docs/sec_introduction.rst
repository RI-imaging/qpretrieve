Citing qpretrieve
=================
If you are using qpretrieve in a scientific publication, please
cite it with:

::

  (...) using qpretrieve version X.X.X (available at
  https://pypi.python.org/pypi/qpretrieve).

or in a bibliography

::
  
  Paul MÃ¼ller (2022), qpretrieve version X.X.X: Phase image analysis
  [Software]. Available at https://pypi.python.org/pypi/qpretrieve.

and replace ``X.X.X`` with the version of qpretrieve that you used.


Furthermore, several ideas implemented in qpretrieve have been described
and published in scientific journals:

- Phase retrieval from holographic images with a gaussian
  filter is implemented according to :cite:`Schuermann2015`.
